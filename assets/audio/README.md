# Flip Sound Setup

1. Download this sound from Pixabay:
   https://pixabay.com/sound-effects/page-flip-smaller-page-453027/
2. Save the MP3 file as:
   page-flip-smaller-page-453027.mp3
3. Place it in this folder:
   assets/audio/

The reader in view.php will use this file for page flip audio.
If the file is missing, it falls back to generated rustle sound.
